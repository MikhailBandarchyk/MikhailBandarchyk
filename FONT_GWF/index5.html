Забавные значения тега font

Почему цвета CANDY и BruceWillis отображаются именно в таком виде?

Дело в том, как считывается HEX-код. Все сиволы, которые не являются символами шеснадцатиричной системы исчисления заменяются на нуль, далее весь код дополняется нулями до числа, кратного трём,затем делится на три части, и отбрасывваются аргументы таким образом, чтобы от каждой из трёх частей остались только два перых символа. Из трёх образовавшихся пар символов и cкладывается код получаемого цвета.
Если применить эти правила, то получим следующее для BruceWillis:

  1. Заменяем все неправильные шестнадцатеричные символы нулями 0, #BruceWillis превращается в b00ce000000
  2. Добавляем необходимое количество нулей до количества знаков, делимого на 3 (11 -> 12), получаем b00ce0000000
  3. Разбиваем на три группы, каждая из которых отвечает за одну компоненту RGB: RGB (b00c, e000, 0000)
  4. Обрезаем каждый из аргументов справа, оставляя 2 символа, и получаем наш результат RGB (b0, e0, 00) = #b0e000 или он же RGB(176, 224, 0).

Применим эти правила и для CANDY:

  1. Заменяем все неправильные шестнадцатеричные символы нулями 0, #CANDY превращается в ca0d0
  2. Добавляем необходимое количество нулей до количества знаков, делимого на 3 (5 -> 6), получаем ca0d00
  3. Разбиваем на три группы, каждая из которых отвечает за одну компоненту RGB: RGB (ca, 0d, 00)
  4. Получаем наш результат RGB (ca, 0d, 00) = #ca0d00 или он же RGB(202, 13, 0).
